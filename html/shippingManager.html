Printing a Corefact label on the shipping manager

On the shipping manager (Click here to access the shipping manager), there's a HELP page that explains the features of the shipping manager in detail (Click here to access the help page). Please reference that page if needed.

Printing a label by searching the agent name

Enter the name of the agent into the SEARCH form. A drop down menu will appear. Select the AGENT NAME option. When the order(s) display, all of the orders are selected by default. If you DON'T want to print labels for all the orders displayed, select the desired order numbers individually. When you're ready to print labels, hover over the ALL ACTIONS button and select PRINT LABELS. A PRINT LABELS window will appear. Confirm accuracy and select PRINT to print all labels for orders in the print labels window.

Printing a 3rd party label

Orders that require a 3rd party label will display a FedEX account number under the ACCT# column. Under the PACKAGEcolumn, the carrier has to be changed to FedEX for the label to print.

Printing a label for a combined shipment

Search for the order(s) using one of the previously mentioned methods above. Hover over the ALL ACTIONS button and select COMBINE SHIPMENTS. A COMBINE ORDERS window will appear. Confirm accuracy and select COMBINE to print a label combining all orders in the combine orders window.
