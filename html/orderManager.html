How to print
Corefact shipping labels

  • Spreadsheet
  • Order manager
  • Shipping manager

Printing a Corefact label on the order manager

On the order manager (Click here to access the order manager), copy and paste the order number into the SEARCH FORM. When the order displays, click on the order number to access the DELIVERY tab. Within the delivery tab, below the SHIPPING STATUS, select CREATE SHIPMENT. A CREATE SHIPMENT window will appear, confirm the PAYOR is Corefact, the WEIGHT for the order is accurate and verify that the correct RATE/CARRIER/DELIVERY OPTION is selected (select the correct rate if necessary). When confirmed, print label by selecting CREATE. Once the label prints, return to the delivery tab and check the SEND CONFIRMAION box followed by UPDATE. This will update the order status to delivered and send the client a delivery confirmation email.

Printing a USPS label

On the CREATE SHIPMENT window, select the PACKAGE drop down menu and select the corresponding flat rate package for the order. Confirm that the rate is correct (select the correct rate if necessary) and print the label by selecting CREATE.

Printing a 3rd party label

On the CREATE SHIPMENT window, select the PAYOR drop down menu and select the corresponding 3rd party account for the order. Confirm that the carrier/delivery option is selected (select the correct rate if necessary) and print the label by selecting CREATE.
